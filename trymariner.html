Getting started with CBL-Mariner

CBL-Mariner (or Mariner) is a distribution of Linux used internally at Microsoft and is intended for use in Azure focused cloud scenarios. The images below are provided without support.

ISO Download

Use this option to download an ISO to install CBL-Mariner in a virtual machine or other test environment.

CBL-Mariner Base Containers

Want to try CBL-Mariner, but don't want to run a full VM? Use the following command to run CBL-Mariner as a container

docker pull cblmariner.azurecr.io/base/core:1.0
